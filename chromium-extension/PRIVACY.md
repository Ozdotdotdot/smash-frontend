This extension does not collect, store, or transmit any personal data.

What it does:
- Runs as a content script on `start.gg` tournament event/bracket pages.
- Injects a button that links the current page to `smash.watch/dashboard` with the page URL as a query parameter.

What it does **not** do:
- No analytics, tracking, or usage data collection.
- No access to cookies or credentials.
- No network requests beyond opening the `smash.watch` dashboard tab when you click the button.
